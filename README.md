# Aaliyah's African Literature Store

An e-commerce landing page dedicated to promoting and selling African literature.

## Problem
African literature is often underrepresented on mainstream e-commerce platforms.
Readers looking specifically for African authors and stories struggle to find a
centralized, accessible marketplace. This makes it difficult for African writers
to reach a global audience and for readers to discover authentic African stories.

## Solution
Aaliyah's African Literature Store is an online store designed specifically for African literature.
It provides a platform where users can browse, search, and purchase books
by African authors. The platform not only highlights African voices but also
connects readers with meaningful stories from the continent.

## Features
- Browse and search books by author or genre
- See the book prices and the pictures of the books
- Featured sections highlighting African authors and new releases  
- Responsive and mobile-friendly design
- Implement user reviews and ratings for books
